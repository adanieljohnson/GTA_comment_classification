Analyzing TA Comments on Student Writing in R

Background

I am interested in understanding how students learn technical writing skills in biology, and how we can use data science methods to improve that process.

Our NSF-sponsored project asks if and how scripted instruction, automated support, and holistic feedback accelerates student development as writers. One of our specific aims is to see if changing the structure and focus of instructors’ comments affects how students’ writing develops over time. To do this, we need to extract instructor comments from student reports, classify those comments, and correlate the types of comments with student performance.

 

Prior Work

In Summer 2018, we extracted and tabulated ~11,000 TA comments from ~500 pairs (initial submissions and revised version) of short lab reports written by undergraduates in 3 different general biology lab courses collected during a single semester. Using an iteratively generated codebook, all 11,000 comments were tagged in two categories:

  • Subject: did the comment focus mainly on Basic criteria, Technical issues, Writing quality, Logic, or Other issues
  • Structure: was the comment mainly Copy correction, General or Specific information, Pointing to another resource, or Holistic coaching for improvement?

 

Counts of the coded TA comments were sorted into a two-axis contingency table. This table provided two pieces of information:

  • A snapshot of the most common types of comments TAs made, and
  • A relative estimate of where TAs are placing their greatest effort.

 

Specific Project Goals

TA comments and meta-data for each student report can be extracted and compiled into a CSV datafile using an existing Unix script. However routinely classifying TA comments by hand is impractical; 10-12,000 comments need to be scored EACH SEMESTER. An automated classification method is needed both for speed and consistency.

For my project I will be building an R-based text analysis workflow that:

  1. Creates a de-identified dataset. Required data will be extracted from the master CSV data set and organized in an anonymized data frame where all names, comments, and confidential metadata are re-keyed to unique random identifiers.
  2. Classifies TA comments according to our existing codebook. The goal is to be able to assign instructor comments into our pre-defined Subject and Structure codes with 90% accuracy or greater. Initially I will be evaluating whether terms frequencies can be used to classify comments. Other methods will be evaluated as they come to light.
  3. Sorts TA comments using unsupervised grouping methods. The codebook for classifying TA comments makes prior assumptions about most relevant text features. Unsupervised learning may uncover more robust or informative classification groups.
